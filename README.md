
# Instalation

```shell
$ git clone https://github.com/LeonardoGodoy/indexer
```

```shell
$ cd indexer
```

```shell
$ make
```

#### Example
##### Options
Run the indexer to see the options available

```shell
$ ./indexer
```

###### Frequent words

```shell
$ ./indexer -f 10 101.txt
```

###### Word frequency

```shell
$ ./indexer -fw it 101.txt
```

###### Document relevance

```shell
$ ./indexer -s it 101.txt 102.txt
```
